# Dark AI SEO Landing Page

Modern AI Startup Landing Page built with **Next.js**, **Framer Motion**, and **Tailwind CSS**. Designed for AI startups, SaaS platforms, and technology businesses with a sleek dark interface, responsive layouts, and smooth animations.

## Features

* **Modern AI startup UI** with a premium dark theme
* **Fully responsive** design for desktop, tablet, and mobile
* **Next.js 14** with React for high performance
* **Tailwind CSS** for scalable utility-first styling
* **Framer Motion** for smooth animations and transitions
* **Reusable components** for easy customization
* **SEO-friendly** project structure
* **Optimized fonts and assets** with Next.js

## Installation

1. **Clone the repository**

```bash
git clone https://github.com/AymaneMehdi/Dark-AI-SEO-Landing-Page.git
```

2. **Navigate to the project**

```bash
cd Dark-AI-SEO-Landing-Page
```

3. **Install dependencies**

```bash
npm install
```

---

## Running the Project

```bash
npm run dev
```

Runs the development server at:

```
http://localhost:3000
```

Open your browser and start editing `src/app/page.tsx`. Changes will automatically reload during development.

---

## Building for Production

```bash
npm run build
npm start
```

Builds the project for production and starts the production server.
